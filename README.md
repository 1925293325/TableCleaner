# TableCleaner

一个简单粗暴的 Typecho 数据库清理工具，专门用来删那些卸载插件后残留的表。

## 干嘛用的

经常用 Typecho 的都知道，很多插件卸载后数据库表还在那儿躺着，时间长了特别烦。这个插件就是帮你一键清理这些残留表的。

**注意**：只清理非核心表（也就是那些插件创建的表），不会动 Typecho 自带的 contents、comments 这些。

## 功能

- 自动扫描数据库里所有非核心表
- 显示每张表有多少条数据
- 智能识别是哪个插件留下的（根据表名关键词猜的，不一定准）
- 查看表结构和前50条数据（确认一下是不是真的要删）
- 一键清空表数据
- 一键删除整个表
- 所有危险操作都有确认弹窗，防止手滑

## 安装

1. 下载解压，文件夹重命名为 `TableCleaner`
2. 扔到 `/usr/plugins/` 目录下
3. 后台启用
4. 在 控制台 -> 数据库表清理 里使用

**不需要配置数据库**，直接用 Typecho 的数据库连接。

## 使用

进去就能看到所有非核心表的列表：

- **空表**：显示绿色，可以直接删
- **有数据**：显示黄色，建议先点"查看内容"确认一下
- **误删风险**：操作前都有 confirm 弹窗，点确定才会执行

### 安全机制

- 表名合法性校验（防止 SQL 注入）
- 核心表白名单保护（contents、comments、users 等绝对删不掉）
- 带 CSRF Token 验证
- 操作后自动跳转并显示结果

## 警告

**删表是不可逆操作！** 虽然插件做了保护，但建议操作前还是备份一下数据库，万一呢。

特别是那个"清空数据"和"删除表"按钮离得挺近的，别点错了。
